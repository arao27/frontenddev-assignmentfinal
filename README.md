# frontenddev-assignmentfinal

## Project Overview
Lift Rank MVP is a React-based web application designed to track your lifts, visualize muscle engagement, and monitor progress. This project demonstrates a professional React architecture with context-based state management, routing, and component composition.

## Features
- Multi-page routing using React Router
- Global state management via React Context API
- User-friendly stat input forms
- Muscle map visualization (placeholder for MVP)
- Saved stats display
- Login page (MVP placeholder)
- Responsive design and clean UI

## Technology Stack
- React 18
- React Router v6
- Context API for state management
- Vite for development and build
- JavaScript & JSX
- CSS for styling
